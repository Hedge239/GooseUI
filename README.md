Latest Release: n/a | License [MIT](https://github.com/Hedge239/GooseUI/blob/main/LICENSE.txt)

Previously named "ZephyrUI", GooseUI is a open-source C++ library. 

## What is GooseUI
GooseUI is designed to simplify the creation of native-feeling applications. It utilizes a graphical renderer (Vulkan or OpenGL) for widget rendering and integrates with native libraries when needed.
